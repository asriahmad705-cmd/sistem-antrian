========================================
  SISTEM ANTRIAN - CARA HOSTING DI RAILWAY
========================================

LANGKAH 1 — Upload ke GitHub
   - Buat akun di github.com (gratis)
   - Klik "New Repository"
   - Upload semua file ini ke repo tersebut

LANGKAH 2 — Deploy ke Railway
   - Buka railway.app
   - Login pakai akun GitHub
   - Klik "New Project" → "Deploy from GitHub repo"
   - Pilih repo kamu

LANGKAH 3 — Tambah MySQL
   - Di dashboard Railway, klik "+ New"
   - Pilih "Database" → "MySQL"
   - Railway otomatis konek ke project kamu

LANGKAH 4 — Buat Tabel (SEKALI SAJA)
   - Di Railway, klik project Flask kamu
   - Buka tab "Settings" → "Railway Shell"
   - Ketik: python setup_db.py

LANGKAH 5 — Generate Domain
   - Klik project Flask kamu
   - Tab "Settings" → "Generate Domain"
   - Kamu dapat URL gratis!
